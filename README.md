# 🎯 Proyek Pemrograman Website 

Repositori ini berisi source code untuk website **MoodSpace**, sebuah platform web interaktif yang dirancang mengikuti suasana hati pengguna dengan kombinasi musik, video, dan quotes berdasarkan karakter emosi dari film **Inside Out**.

Proyek ini dibuat menggunakan HTML, CSS, dan JavaScript dengan fokus pada *Dynamic Rendering* dan *Responsive Design*.

## 📂 Struktur Repositori

```text
assets/               # Folder aset media
├── logo.png          # Logo MoodSpace
├── joy/              # Aset khusus Joy
├── sadness/          # Aset khusus Sadness
├── anger/            # Aset khusus Anger
├── ...               # Folder mood lainnya
├── mood              # Icon Mood
└── profil/           # Foto anggota kelompok
index.html            # Halaman Utama (Tempat Memilih Karakter)
music.html            # Berisi Musik sesuai mood dengan format spotify playlist dan local musik
video.html            # Berisi Video sesuai mood
quotes.html           # Berisi Kata Kata sesuai mood
style.css             # Styling & Layout 
script.js             # Database Konten & Logika Website
```

## 🛠️ Persyaratan Sistem

1. Memiliki Web Browser Modern (Google Chrome, Microsoft Edge, Firefox, Safari)

2. Memiliki Text Editor (VS Code, Sublime Text, atau Notepad++) 

3. Koneksi Internet untuk memuat Font Awesome & Spotify Embed

4. Git untuk mengelola kode

## 🚀 Memulai

Pastikan Anda memiliki Git dan Web Browser yang terinstall.

Clone repositori ini:

```bash
git clone [https://github.com/GabzyP/Project-Praktikum-Web-Mood-Space.git](https://github.com/GabzyP/Project-Praktikum-Web-Mood-Space.git)
```

Masuk ke direktori proyek:
```bash
cd Project-Praktikum-Web-Mood-Space
```

## Menjalankan Aplikasi:

1. Cari file bernama index.html

2. Klik dua kali file tersebut untuk membukanya di browser.

3. Atau jika menggunakan VS Code, pilih index.html lalu klik run atau klik shorcut keyboard **F5**.

## 📚 Fitur Proyek MoodSpace

Fitur Utama:

-  Konsol Mood: Memilih 9 karakter Inside Out (Joy, Sadness, Anger, dll) yang mengubah tema warna website secara otomatis.

-  Cinema Room: Pemutar video dinamis dengan layout Video Utama dan List Video disamping yang menyesuaikan konten dengan mood.

-  Music Room: Integrasi Spotify Embed dan Local Music Player dengan tampilan kartu album estetik.

-  Quotes: Galeri kutipan inspiratif dengan fitur Tambah Quote Sendiri.

-  Responsive Design: Tampilan menyesuaikan layar Desktop, Tablet, dan Mobile.
